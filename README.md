# 🏝️ Treasure Island Game

A simple text-based adventure game built with Python as part of the **100 Days of Code: The Complete Python Pro Bootcamp** by Dr. Angela Yu.

## 📌 Project Description

Treasure Island is a command-line adventure game where the player makes a series of choices to find the hidden treasure. Each decision leads to a different outcome, making it an interactive beginner-friendly Python project.

## 🎮 Gameplay

The player goes through the following choices:

1. Choose a direction:
   - Left
   - Right

2. Reach a lake:
   - Wait
   - Swim

3. Choose a door:
   - Red
   - Yellow
   - Blue

### 🏆 Winning Path

```
Left ➜ Wait ➜ Yellow
```

Any other choice results in **Game Over**.

## 🛠️ Technologies Used

- Python 3
- Conditional Statements (`if`, `elif`, `else`)
- User Input (`input()`)
- Print Statements
